# 🛒 Shopping List Bot

Телеграм-бот для совместного ведения списка покупок. Работает 24/7 в облаке, поддерживает пакетное добавление товаров и безопасный доступ.

## ✨ Особенности

- **Пакетное добавление:** отправьте список через запятую (напр., `молоко, хлеб, сыр`), и бот превратит их в отдельные пункты.
- **Интерактивный список:** нажимайте на кнопку с товаром (✅), чтобы мгновенно вычеркнуть его.
- **Безопасность:** доступ разрешен только по списку Telegram ID пользователей.
- **Облачное хранение:** данные хранятся в PostgreSQL (Neon), они не пропадут после перезагрузки.
- **24/7 Работа:** развернут на Render с автоматическим деплоем из GitHub.

## 🛠 Технологии

- **Библиотека бота:** [pyTelegramBotAPI](https://github.com/eternnoir/pyTelegramBotAPI)
- **База данных:** [PostgreSQL](https://www.postgresql.org/) (хостинг [Neon](https://neon.tech/))
- **Хостинг:** [Render](https://render.com/)
- **Веб-сервер:** Flask (для Health Checks)

## 🚀 Настройка и запуск

### Переменные окружения

Для работы на сервере (Render/Docker) нужны следующие переменные:
- `API_TOKEN`: Токен от @BotFather.
- `DATABASE_URL`: Строка подключения к базе данных.
- `ALLOWED_USERS`: Список ID через запятую (напр., `123456,789012`).

### Локальный запуск
1. Установите зависимости: `pip install -r requirements.txt`
2. Создайте файл `.env` на основе переменных выше.
3. Запустите: `python main.py`

## 📝 Команды
- `/start` или `/help` — меню.
- `/list` — показать список.
- `/clear` — очистить всё.
