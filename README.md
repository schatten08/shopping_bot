# Shopping List Telegram Bot

Простой и удобный бот для управления списком покупок. Список сохраняется в файл, так что данные не пропадут при перезагрузке.

## Функционал
- Добавление товаров простым текстовым сообщением.
- Просмотр списка с кнопками для удаления (вычеркивания) товаров.
- Полная очистка списка.
- Постоянное хранение данных в JSON.

## Установка

1. Клонируйте репозиторий или скопируйте файлы.
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Получите токен у [@BotFather](https://t.me/BotFather).
4. Вставьте ваш токен в файл `main.py` в переменную `API_TOKEN`.

## Запуск
```bash
python main.py
```
