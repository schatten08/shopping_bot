# Changelog

Все важные изменения в проекте Shopping List Bot будут фиксироваться здесь.

## [1.2.0] - 2026-06-08
### Добавлено
- Система контроля доступа: теперь бот отвечает только пользователям из списка `ALLOWED_USERS`.
- Поддержка добавления нескольких товаров одной строкой через запятую.
- Дизайн списка: замена текстовых пометок на интерактивные кнопки с эмодзи ✅.

## [1.1.0] - 2026-06-08
### Добавлено
- Интеграция с PostgreSQL (Neon) для надежного хранения данных.
- Файл конфигурации `Procfile` и Flask-сервер для деплоя на Render.
- Команды `/list` и `/clear` для быстрого управления списком.
- Автоматическая настройка меню команд в Telegram.

## [1.0.0] - 2026-06-08
### Добавлено
- Базовый функционал бота: добавление товаров и удаление через Inline-кнопки.
- Локальное хранение данных в JSON.
